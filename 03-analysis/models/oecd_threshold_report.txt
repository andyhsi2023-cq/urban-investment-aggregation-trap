  城市资本-产出比 (CPR) 开始系统性下降,表明建设投资效率出现转折。

总结
========================================================================

1. 数据获取:
   OECD API 不可用，使用文献基准的建设投资/GFCF 比例代理变量
   代理变量来源: OECD National Accounts Table 8A + 国别文献
   限制: 假设了固定的建设投资占比，未反映年度波动

2. 新阈值估计 (Construction/GDP):
   gamma_hat = 9.1%
   Bootstrap 95% CI = [8.1%, 17.6%]
   CI 宽度 = 9.5pp
   F 检验: F = 10.31, p = 0.000034

3. 与原 GFCF/GDP 阈值比较:
   原 GFCF/GDP 阈值: ~15% (CI: [15%, 31%], 宽度 16pp)
   新 Construction/GDP 阈值: 9.1% (CI: [8.1%, 17.6%], 宽度 9.5pp)
   CI 是否缩窄: 是 (改善 6.5pp)

4. 分收入组稳定性:
   Low income: gamma = 15.8% [6.3, 16.3]
   Lower middle income: gamma = 13.0% [9.0, 18.0]
   Upper middle income: gamma = 9.5% [9.0, 18.0]
   High income: gamma = 14.1% [9.6, 15.6]
   CI 重叠对数: 6

5. 中国定位:
   首次突破建设投资阈值: 1960
   当前超出程度: 20.4pp

6. 方法论说明:
   - 由于 OECD API 数据获取限制，本分析使用代理变量方法
   - 建设投资/GFCF 比例基于 OECD 已发布数据和实证文献
   - 建议: 如有条件获取 OECD.Stat 原始数据，应替换代理变量重跑
   - 该代理方法的核心假设 (建设占比在国家内相对稳定) 在长期趋势中
     基本成立，但短期波动 (如西班牙2000-2008房地产泡沫) 会被低估
