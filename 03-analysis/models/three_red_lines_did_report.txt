10. 结果解读
======================================================================

主效应 ln(HP): beta = -0.0059, p = 0.683673
主效应 Q:      beta = -0.0632, p = 0.006736

--- 因果假说检验 ---

假说 A ("过度投资有害"): 限制投资后，高依赖城市 Q 应止跌/回升
  预测: beta_DID(Q) > 0

假说 B ("经济冲击"): 政策冲击直接打压高依赖城市房价
  预测: beta_DID(Q) < 0, beta_DID(HP) < 0

结论 [ln(HP)]: DID 交互项不显著 (p = 0.6837)
  点估计方向: 负, 但统计上不能区分两个假说

结论 [Q]: DID 交互项显著为负 (p < 0.05)
  高依赖城市政策后 Q 跌幅更大 — 支持假说B
  然而, 这并不排除假说A。两种机制可能同时存在:
  短期内政策冲击打压需求(V下降)和投资(K不变) -> Q下降
  长期效应需要更长的观察窗口才能显现
