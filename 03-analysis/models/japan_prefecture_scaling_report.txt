步骤 3: 標度律回帰分析
========================================================================

【Model 1】人均GDP標度律: ln(GDP_pc) ~ α·ln(Pop)
  如果 α > 0 → 超線形標度 (agglomeration economies)
  如果 α = 0 → 線形標度
  如果 α < 0 → 亜線形標度

--- ln(GDP_per_capita) ~ ln(Population) ---
  α (slope)     = 0.0804  (SE = 0.0495)
  95% CI        = [-0.0166, 0.1774]
  p-value       = 1.04e-01
  R²            = 0.1417
  F-stat p      = 1.11e-01
  N             = 47
  Jarque-Bera p = 0.0003
  Shapiro-Wilk p= 0.0160

【Model 2】人均住宅標度律: ln(Housing_pc) ~ β·ln(Pop)
  如果 β < 0 → 大都市人均住宅少 (housing scarcity scaling)

--- ln(Housing_per_capita) ~ ln(Population) ---
  α (slope)     = 0.0226  (SE = 0.0142)
  95% CI        = [-0.0052, 0.0505]
  p-value       = 1.11e-01
  R²            = 0.0680
  F-stat p      = 1.18e-01
  N             = 47
  Jarque-Bera p = 0.8098
  Shapiro-Wilk p= 0.7582

【Model 3】空置率標度律: ln(Vacancy%) ~ γ·ln(Pop)
  如果 γ < 0 → 小県空置率高 (OCR scaling)
  这直接对应 OCR 概念：小城市过度建设

--- ln(Vacancy_rate) ~ ln(Population) ---
  α (slope)     = -0.0939  (SE = 0.0233)
  95% CI        = [-0.1396, -0.0481]
  p-value       = 5.76e-05
  R²            = 0.2577
  F-stat p      = 2.18e-04
  N             = 47
  Jarque-Bera p = 0.8014
  Shapiro-Wilk p= 0.8829

【Model 4】空置率 vs 人口密度: ln(Vacancy%) ~ δ·ln(Pop_density)

--- ln(Vacancy_rate) ~ ln(Pop_density) ---
  α (slope)     = -0.0817  (SE = 0.0213)
  95% CI        = [-0.1236, -0.0399]
  p-value       = 1.28e-04
  R²            = 0.3336
  F-stat p      = 3.93e-04
  N             = 47
  Jarque-Bera p = 0.4573
  Shapiro-Wilk p= 0.5088

步骤 6: 日本的Q<1周期与空間格差
========================================================================

日本是唯一经历完整 Q<1 周期的发达经济体:
  - 1990年バブル崩壊後、不動産価値 < 再建築費 が30年間持続
  - 全国空置率 2018年: 13.6%（846万戸）
  - 人口減少は地方から進行 → 空置率の地域格差拡大

空置率 vs 人口 (Spearman): rho = -0.4775, p = 0.0007
大都市圏 (≥200万人, N=16): 平均空置率 = 13.5%
地方県 (<100万人, N=10): 平均空置率 = 16.1%
  Welch t-test: t = -3.03, p = 0.0058

理論的含意:
  空置率は過剰建設率 (OCR) の直接指標
  ln(Vacancy%) ~ γ·ln(Pop) で γ < 0 ならば:
  → 小都市ほど住宅過剰 (OCR高い)
  → Urban Tobin's Q フレームワークと整合
  ★ 日本データは γ = -0.0939 < 0 → 仮説支持

========================================================================
步骤 9: 総括 — 日本都道府県標度律
========================================================================

主要発見:
  1. 人均GDP標度指数 α = 0.0804 (p = 1.04e-01)
     → 正の傾向あるが統計的に有意でない
  2. 人均住宅標度指数 β = 0.0226 (p = 1.11e-01)
  3. 空置率標度指数 γ = -0.0939 (p = 5.76e-05)
     → 小県ほど空置率高い → OCR標度律成立
     → Urban Tobin's Q理論と整合: Q<1環境で小都市が最も過剰

理論的含意 (Urban Q Phase Transition):
  日本のデータは以下を示唆:
  - バブル崩壊後のQ<1長期化は全国的現象だが、影響は均一でない
  - 集積の経済が弱い地方県ほど、過剰建設の影響が深刻
  - 空置率の地域格差は「相転移の空間的異質性」を表す
  - これは中国の将来予測に直接応用可能
