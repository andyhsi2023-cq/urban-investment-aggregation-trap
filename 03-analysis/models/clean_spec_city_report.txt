--- Beta 衰减比 (Attenuation Ratio) ---
  定义: 1 - (clean_beta / original_beta)
  含义: clean spec 下 beta 相对于原始 spec 衰减了多少
         衰减部分 = 可归因于机械相关的份额

  中国城市:
    原始 beta (MUQ ~ FAI/GDP) = -2.2571
    Clean beta (DeltaV/GDP ~ FAI/GDP) = -0.3669
    衰减比 = 1 - (-0.3669 / -2.2571) = 0.8374
    机械相关份额 = 83.7%
    残余真实效应份额 = 16.3%

  Clean → City FE 比较:
    Clean beta (pooled) = -0.3669
    City FE beta = 0.5210
    >> 符号翻转: pooled 为负, city FE 为正
    >> 这表明 pooled OLS 的负效应完全由城市间 (between) 变异驱动
    >> 控制城市固定效应后, 城市内 (within) 变异方向相反

  总衰减 (原始 → TWFE):
    TWFE beta = 0.1560
    总衰减比 = 1 - (0.1560 / -2.2571) = 1.0691
    原始 beta 中 106.9% 来自机械相关+城市/年份异质性

PANEL G: 核心结论与审稿人回应要点
================================================================================

1. 机械相关量化:
   原始 beta (MUQ~FAI/GDP) = -2.2571
   Clean beta (DeltaV/GDP~FAI/GDP) = -0.3669
   衰减比 = 83.7% → 原始 beta 中约 83.7% 可归因于机械相关
   但 clean spec 下 beta 仍然显著为负 (p = 1.935483e-02)

2. Within-estimator (城市固定效应):
   City FE beta = 0.5210, p = 3.438340e-07
   TWFE beta = 0.1560, p = 4.684430e-01
   >> 关键发现: City FE 下符号翻转 (负→正)
   构成 Simpson's Paradox:
     Between (城市间): 投资比高的城市, 价值增量/GDP低 (负)
     Within (城市内): 同一城市投资比上升时, 价值增量/GDP也上升 (正)
   解读: 城市间系统性差异是核心 -- 长期过度投资的城市效率低
         短期内投资增长可能反映经济景气共同趋势
   面板结构限制: 150/213 城市仅1期观测, within variation 极有限

3. Sign reversal (中美对比):
   Clean spec 下: 中国 beta = -0.3669 (负), 美国 beta = 2.8097 (正)
   Sign reversal 在消除机械相关后 仍然成立

4. 人口加权统计:
   2016年 MUQ<1 城市: 82.2% (未加权), 70.2% (人口加权)

5. 审稿人回应建议:
   - 将 clean spec (DeltaV/GDP~FAI/GDP) 作为新的主规范
   - 原始 MUQ 回归保留在 Extended Data 中, 标注机械相关
   - 在 Methods 中增加衰减比分析
   - City FE 不显著问题: 解释为城市间（Between）变异是主要识别来源
   - 建议增加: 工具变量策略 (如用历史投资率作为 IV)
