======================================================================
中国 Urban Q 多口径计算报告
基于国家统计局 / 财政部真实数据
======================================================================

一、数据来源
----------------------------------------
  住宅均价: NBS《中国房地产统计年鉴》, 1998-2024
  竣工面积: NBS《中国统计年鉴》, 1998-2024 (1985-1997为估算)
  销售面积/金额: NBS, 1998-2024
  房地产投资: NBS《中国统计年鉴》, 1987-2024
  基础设施投资: NBS《中国统计年鉴》, 1990-2024
  土地出让收入: 财政部《全国政府性基金收入》, 1999-2024
  GDP: NBS《中国统计年鉴》, 1978-2024
  建成区面积: NBS, 1981-2024

二、方法论
----------------------------------------
  Urban Q = V(t) / K(t)
  V: 城市资产市场价值; K: 累计建设投资重置成本

  V1 = 住宅均价 x PIM住宅存量 (delta=2%)
  V2 = 累计商品房销售额
  V3 = V1 + 累计土地出让 + PIM基础设施存量 (delta=4%)

  K1 = PIM(房地产投资, delta=2%)
  K2 = PIM(房地产+基础设施投资, delta=3%)
  K3 = PIM(全口径建设投资, delta=3%)

  MUQ = dV/dI (边际Urban Q)

四、推荐口径分析
----------------------------------------

  [推荐1] Q_V1K1 = 住宅市值 / 房地产投资存量
  含义: 住宅值多少 vs 花了多少建住宅
  1998: 4.940
  峰值: 4.940 (1998)
  2024: 1.278

  [推荐2] Q_V1K2 = 住宅市值 / (房地产+基础设施)投资存量
  含义: 住宅价值能否覆盖全部城市建设投入
  1998: 1.778
  峰值: 1.778 (1998)
  2024: 0.580

  [推荐3] Q_V3K2 = 综合资产价值 / 总建设投资存量
  含义: 城市全部资产价值 vs 全部建设投入
  1998: 2.413
  峰值: 2.413 (1998)
  2024: 1.375

七、分阶段特征
----------------------------------------
  1998-2003 (房改初期):
    Q_V1K1均值=3.916, MUQ_V1均值=2.061, 房地产投资/GDP=5.5%
  2004-2007 (快速扩张):
    Q_V1K1均值=3.023, MUQ_V1均值=2.667, 房地产投资/GDP=8.7%
  2008-2009 (全球金融危机):
    Q_V1K1均值=2.633, MUQ_V1均值=2.069, 房地产投资/GDP=10.0%
  2010-2013 (刺激与过热):
    Q_V1K1均值=2.304, MUQ_V1均值=1.555, 房地产投资/GDP=13.0%
  2014-2016 (调整与去库存):
    Q_V1K1均值=1.899, MUQ_V1均值=1.361, 房地产投资/GDP=14.1%
  2017-2019 (棚改与调控):
    Q_V1K1均值=1.842, MUQ_V1均值=1.437, 房地产投资/GDP=13.2%
  2020-2021 (疫情与尾声):
    Q_V1K1均值=1.706, MUQ_V1均值=0.862, 房地产投资/GDP=13.4%
  2022-2024 (下行与转型):
    Q_V1K1均值=1.385, MUQ_V1均值=-0.418, 房地产投资/GDP=9.1%

八、核心结论
----------------------------------------
  1. Urban Q 从峰值大幅回落，显示投资回报率急剧下降
  2. 边际 Urban Q (MUQ) 在下行期转负，
     说明新增投资在毁灭而非创造价值
  3. 这些证据支持 expansion -> renewal 的相变叙事:
     Q > 1 时理性选择是扩张建设
     Q < 1 时理性选择是存量更新
