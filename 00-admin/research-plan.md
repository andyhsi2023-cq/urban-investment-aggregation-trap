## 1. 研究背景与问题

### 1.1 核心问题

城市化进入成熟期后，城市经济从扩张驱动转向更新驱动。这一转型在全球已完成城镇化的国家普遍发生，但至今缺少：
- 一个**统一的理论框架**解释为什么转型必然发生
- 一个**可计算的临界指标**判断转型何时发生
- 一套**经济学决策准则**指导转型期的投资与管理

### 1.2 理论缺口

- 城市经济学：大量描述性研究，缺乏类似 Tobin's Q 的投资决策理论
- 城市更新：政策导向文献丰富，经济学建模薄弱
- 城市科学：标度律 (West & Bettencourt) 关注截面特征，较少关注生命周期转换

### 1.3 本研究的理论贡献

提出 **Urban Q**（城市 Tobin's Q），将金融学的资产定价逻辑移植到城市经济：

> Urban Q = 城市资产市场价值 / 累计建设投资（重置成本）

并论证 Urban Q = 1.0 是城市经济的**相变临界点**：
- Q > 1：扩张相——投资创造超额价值，理性行为是继续扩建
- Q < 1：更新相——投资边际价值不足，理性行为是管理存量
- 相变不可逆：无国际先例显示成熟经济体重新回到 Q > 1

## 2. 研究目标

### 2.1 主要目标
构建 Urban Q 理论模型，实证检验其作为城市扩张-更新相变临界值的普适性。

### 2.2 次要目标
1. 设计比率曲线体系（N/R ratio, I/V ratio, Flow/Stock ratio, 产业结构曲线），实现跨国可比的城市生命周期诊断
2. 识别 Urban Q = 1 时的伴随信号群（临界城镇化率、产业结构阈值等）
3. 构建城市更新经济学决策框架，为立项-投资-评审-运营全流程提供准则
4. 检验"相变不可逆性"假说

## 3. 理论框架

### 3.1 Urban Q 的经济学推导

#### 从 Tobin's Q 到 Urban Q

Tobin's Q (1969) 的原始定义：
- Q = 企业市场价值 / 资本重置成本
- Q > 1 → 投资扩张；Q < 1 → 不投资

移植到城市：
- 分子：城市建成区的市场价值总量（住宅 + 商业 + 基础设施）
- 分母：重置这些资产的建设成本（累计固定资产投资的折旧调整值）

#### 边际 Urban Q（Marginal Urban Q）

- MUQ(t) = ΔV(t) / ΔI(t) = 边际资产价值增量 / 边际投资增量
- 即 six-curves 项目中的 ICR（投资资本化率）
- MUQ < 1：边际投资在摧毁价值

#### Urban Q 的动态方程

dQ/dt = f(城镇化速度, 产业升级速度, 资产折旧率, 投资强度)

当城镇化减速 + 资产折旧加速 + 投资惯性维持 → dQ/dt < 0 → Q 趋向并突破 1.0

### 3.2 比率曲线体系

| 比率曲线 | 定义 | 经济学含义 | 预期形态 |
|---------|------|-----------|---------|
| **R1: 城镇化率** | 城镇人口/总人口 | 需求侧锚定 | S 曲线 |
| **R2: 新建/更新投资比 (N/R)** | 新建投资 / 更新改造投资 | 投资结构转型 | 单调递减，临界值=1 |
| **R3: 投资/资产价值比 (I/V)** | 年度建设投资 / 城市资产总值 | ≈ 1/Q 的流量版本 | 先升后降再升（U型？） |
| **R4: 新建/存量比 (F/S)** | 年度新建面积 / 累计建筑存量 | 增量贡献率 | 单调递减趋近0 |
| **R5: 产业结构曲线** | 三产占比 | 经济功能转型 | 二产倒U，三产S曲线 |

### 3.3 相变模型

借鉴物理学相变理论（Landau theory）：
- **序参量 (Order Parameter)**：Urban Q
- **控制参量 (Control Parameter)**：城镇化率
- **临界点 (Critical Point)**：Q = 1.0
- **对称性破缺**：扩张相中"多建=多赚"的对称性，在更新相中破缺为"多建=多亏"

### 3.4 城市更新经济学决策框架

| 决策层级 | 核心指标 | 决策规则 |
|---------|---------|---------|
| **国家/区域** | 城市Q值分布图 | 识别已越过临界点的城市群，调整财政转移 |
| **城市层面** | 城市MUQ趋势 | MUQ < 1 时停止新建用地供给 |
| **项目立项** | 项目Q值 | Q > 1 方可立项；Q ∈ (0.8,1) 需特殊论证 |
| **投资规模** | 边际收益率 | 投资回报率 > 城市平均资本成本 |
| **建设运营** | 全生命周期 NPV | 含30年运维成本的净现值 > 0 |
| **资产管理** | 资产折旧率 vs 增值率 | 折旧 > 增值时触发更新决策 |

### Phase 1: Urban Q 计算与特征刻画

- 275城市面板 Urban Q (2005-2019)

- 5条比率曲线的计算与可视化

- Urban Q = 1 的临界城镇化率（中国 + 国际）
- 断点检验（Chow test, Bai-Perron）
- 相变信号群的同步性分析

### Phase 4: 经济学模型构建
- Urban Q 的动态方程推导
- 最优投资规则的解析解
- 城市更新决策框架的参数校准

### Phase 5: 稳健性与预测
- 替代 C5 定义的敏感性分析
- 样本外预测检验（用日本/韩国的历史数据）
- 蒙特卡洛模拟的不确定性量化
