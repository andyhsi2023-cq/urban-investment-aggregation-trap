# Simpson's Paradox Masks Declining Returns on Urban Investment Worldwide

**Replication package for:**

Xi, H. (2026). Simpson's paradox masks declining returns on urban investment worldwide. *Nature* (submitted).

---

## Abstract

Aggregate statistics suggest returns on urban investment remain stable as economies urbanise. This stability is a Simpson's paradox. We construct a marginal Urban Q across 157 countries and 1,567 subnational regions spanning eight countries and six continents. Within every developing-economy income group, returns decline with urbanisation (all p < 0.001 under a GDP-based formulation immune to housing-price cycles), but compositional shifts conceal the decline. We prove this aggregation trap is a mathematical necessity under three empirically verified conditions -- and that it extends to any domain where heterogeneous units are evaluated on pooled metrics. At the city level, investment intensity is negatively associated with returns in China (beta = -0.37) but positively in the United States (+2.81) and Japan (+0.64). Japan's 67-year prefectural record shows that at matched urbanisation rates, China's returns are 3.4 times lower alongside 2.3 times higher investment intensity. The cumulative below-parity investment in China alone totalled approximately US$18 trillion (90% CI: 12--24) between 2000 and 2024.

## Reproducing All Results

### Step 1: Download raw data

```bash
# Automated downloads (World Bank, PWT, BEA, Census, Eurostat)
python analysis/n30_process_raw_data.py --download

# Manual downloads required for:
#   - Japan Cabinet Office SNA (see data/raw/japan_cabinet_office/README.md)
#   - Bank of Korea ECOS (see data/raw/korea_ecos/README.md)
#   - NBS China yearbooks (see data/raw/china_nbs/README.md)
#   - ABS Australia (see data/raw/australia_abs/README.md)
#   - Statistics South Africa (see data/raw/south_africa_statssa/README.md)
```

### Step 2: Process raw data into analysis-ready panels

```bash
python analysis/n30_process_raw_data.py
```

This generates all files in `data/processed/`.

### Step 3: Run all analyses

```bash
# Run all analysis scripts sequentially (approximately 45 minutes)
for script in analysis/n0[1-9]_*.py analysis/n1[0-9]_*.py analysis/n2[0-2]_*.py; do
    echo "Running $script ..."
    python "$script"
done
```

Or run individual scripts:

```bash
python analysis/n01_gdp_muq_simpson.py      # GDP-based Simpson's paradox
python analysis/n08_aggregation_trap_verification.py  # Theorem verification
python analysis/n12_japan_prefecture_clean_spec.py    # Japan 67-year analysis
```

### Step 4: Generate all figures

```bash
python analysis/n40_all_figures.py
```

All figures are saved to `figures/`.

### Random seeds

All stochastic procedures use fixed random seeds for exact reproducibility:
- City-level analyses: `seed = 42`
- Unified panel: `seed = 20260322`
- Monte Carlo simulations: `seed = 20260321`

## Script Descriptions

| Script | Description | Output |
|--------|-------------|--------|
| `n01_gdp_muq_simpson.py` | Computes GDP-based MUQ (= DeltaGDP / GFCF) for 157 countries; tests Simpson's paradox within four World Bank income groups | Spearman rho, p-values per group |
| `n02_housing_muq_simpson.py` | Computes housing-based MUQ for 144 countries using PWT capital stock | Parallel paradox verification |
| `n03_unified_panel.py` | Harmonises 1,567 subnational regions across 8 countries; panel regression with country + year FE | beta = -0.043, p < 0.001 |
| `n04_block_bootstrap.py` | Country-level block bootstrap (2,000 iterations) preserving time-series structure | Bootstrap p-values |
| `n05_leave_one_out.py` | Leave-one-out analysis: removes each of 40 upper-middle-income countries | 40/40 negative correlations |
| `n06_betav_decomposition.py` | Decomposes scaling exponent beta_V = 1 + beta_A + beta_P for China and US | Mechanical (94.6%) vs economic |
| `n07_sur_estimation.py` | Seemingly unrelated regression for cross-national Delta-beta inference | Joint standard errors by year |
| `n08_aggregation_trap_verification.py` | Tests theorem conditions A1--A3 at cross-national level | All three satisfied |
| `n09_within_country_paradox_test.py` | Tests A1--A3 within 7 countries at subnational level | 0/7 satisfy all three |
| `n10_china_city_clean_spec.py` | Clean specification DeltaV/GDP ~ FAI/GDP for 213 Chinese cities | beta = -0.37, p = 0.019 |
| `n11_us_msa_clean_spec.py` | Clean specification for 921 US MSAs | beta = +2.81, p < 10^-6 |
| `n12_japan_prefecture_clean_spec.py` | 47 prefectures, 1956--2022: pooled, FE, TWFE, period decomposition, Bai-Perron | 67-year trajectory |
| `n13_china_japan_mirror.py` | Aligns China and Japan by urbanisation rate; computes MUQ ratio and decline rates | 3.4x lower, 2.7x faster decline |
| `n14_crisis_recovery.py` | Japan post-bubble, Korea 1997, PIIGS: recovery ratios and structural shifts | Three crisis typologies |
| `n15_china_province_fe.py` | Province fixed effects on 31 provinces (2011--2019) | beta = -0.164, p = 0.001 |
| `n16_quantity_price_decomposition.py` | Decomposes DeltaV into new construction vs price appreciation | China 44% vs US 11% quantity |
| `n17_within_city_estimator.py` | City FE and TWFE; documents sign reversal | beta_FE = +0.52 |
| `n18_did_three_red_lines.py` | DID exploiting 2020 credit-tightening policy (reported in ED only) | Diagnostic limitations noted |
| `n19_monte_carlo_calibration.py` | 10,000-iteration joint sampling across 7 calibration variants | Q=1 crossing year 90% CI |
| `n20_bai_perron_breaks.py` | Structural break detection in national and prefectural MUQ series | Breaks at 1980, 1990 (Japan) |

| `n22_robustness_checks.py` | Balanced panels, Newey-West, BH-FDR, spatial autocorrelation, winsorisation | 22/25 tests significant |
