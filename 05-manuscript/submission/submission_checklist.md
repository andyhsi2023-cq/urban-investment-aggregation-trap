## 8. Manuscript Quality Checks

### Language and style
- [ ] First person singular ("I") used consistently (sole author)
- [ ] No hedging language without justification ("It is well known that...", etc.)
- [ ] Active voice preferred throughout
- [ ] All abbreviations defined at first use (MUQ, GFCF, FAI, ICOR, SUR, TWFE, DID)
- [ ] Maximum 3--4 abbreviations in main text
- [ ] Nature reference style (numbered, superscript, in order of appearance)

### Statistical reporting
- [ ] All p-values reported with test statistic
- [ ] Effect sizes with confidence intervals
- [ ] Sample sizes (N) stated for every test
- [ ] Multiple comparison correction noted (BH-FDR: 22/25 significant)
- [ ] Random seeds documented for reproducibility

### Internal consistency
- [ ] All numbers in abstract match main text
- [ ] All numbers in main text match Extended Data tables
- [ ] Figure references match actual figure content
- [ ] Reference numbers sequential and correct
- [ ] No orphaned references (cited but not listed, or listed but not cited)

### Scope and limitations
- [ ] Descriptive (not causal) positioning stated in Introduction and Discussion
- [ ] Within-city sign reversal explicitly discussed as boundary condition
- [ ] Within-country paradox failure explicitly stated
- [ ] Five limitations in Discussion match actual analysis constraints
- [ ] MUQ != social value caveat included
